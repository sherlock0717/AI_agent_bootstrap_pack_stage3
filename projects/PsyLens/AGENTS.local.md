# PsyLens Local Rules

## 项目定位
PsyLens 是当前已经接入系统的样例项目，用于验证系统的项目接入、项目工作流、项目复盘、交付物管理与知识沉淀机制。

## 项目边界
- PsyLens 不是系统母仓库本体。
- PsyLens 不是默认项目模板。
- PsyLens 只是当前已接入的样例项目之一。
- 后续新项目默认应从 C:/Users/22358/Desktop/项目 中创建，并通过系统注册接入。

## 当前作用
当前优先作为以下能力的验证样例：
- external project onboarding
- project note / session / review / snapshot
- deliverables layer
- project operations surface

## 工具优先级
- 代码与脚本修改：优先使用 Cursor
- 复杂调试与规则梳理：可使用 Claude Code
- 系统设计、流程整理、文档规划：可使用 ChatGPT
- 可确定执行的步骤：优先写成脚本，不只停留在对话中

## 输出与归档
- 项目知识记录放在：knowledge/projects/PsyLens/
- 项目输出放在：outputs/projects/PsyLens/
- 项目日志放在：logs/projects/PsyLens/

## 当前限制
当前阶段不把 PsyLens 视为系统默认项目，而只作为接入验证样例。
