# Repository Governance

## Purpose
This repository is the root repository for the reusable AI work system.

## What belongs here
- reusable rules
- reusable scripts
- system-level configuration
- onboarding templates
- knowledge structure
- scout workflows
- maintenance documents

## What should stay outside
- full codebases of external projects
- project-specific large data
- temporary local outputs
- secrets and local credentials

## Current remote
- GitHub origin is configured
- Branch: main

## Maintenance habits
- Start work with start_day.ps1
- Close work with close_day.ps1
- Publish meaningful snapshots with publish_snapshot.ps1
- Keep important structural changes documented in knowledge
